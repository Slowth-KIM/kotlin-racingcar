## 문자열 계산기 테스트 케이스 목록

기본 연산 테스트

- [ ] 덧셈: "2 + 3" = 5
- [ ] 뺄셈: "5 - 3" = 2
- [ ] 곱셈: "4 * 3" = 12
- [ ] 나눗셈: "8 / 2" = 4

예외 처리 테스트

- [ ] null 입력 처리
- [ ] 빈 문자열 입력 처리
- [ ] 잘못된 연산자 입력 처리 ("2 @ 3")
- [ ] 잘못된 형식의 입력 처리 ("2 + ")

복합 연산 테스트

- [ ] 두 개의 연산: "2 + 3 * 4" = 20
- [ ] 세 개의 연산: "2 + 3 * 4 / 2" = 10
- [ ] 동일 우선순위 연산: "1 + 2 + 3" = 6
